<h1>Sobre el proyecto</h1>
<h2>Resumen</h2>
<p>Se trata de un proyecto integrador desarrollado exclusivamente en lenguaje Python con el objetivo de aplicar los conocimientos obtenidos para acceder a la certificación de finalización del curso brindado por ADA IT.</p>
<p>El programa consiste en un generador y editor de catálogos de películas almacenados en archivos .txt, dónde cada archivo se corresponde con un catálogo compuesto por un conjunto de datos de una o más películas. El identificador del catálogo es su nombre y cada película almacenada detalla: título, director, año y review (crítica).</p>
<p>Hasta el momento el programa no cuenta con una interfaz gráfica para su interacción, aunque se planea desarrollar una en el futuro en una nueva versión del mismo.</p>

<h2>Flujos y funcionalidades</h2>

<h3>1. Menú de ingreso</h3>
<p>Al iniciar el programa se muestra un mensaje de bienvenida y un menú compuesto por dos opciones: 1) Ingresar a un catálogo y 2) Salir. También se muestra un input con el texto "Ingrese una opción" para que el usuario acceda a una de las opciones del menú.</p>
<p>En caso de seleccionar la opción 1) Ingresar a un catálogo se muestra al usuario el listado de catálogos existentes y se lo redirige a un nuevo menú (Ver apartado 2. Ingreso al catálogo)</p>
<p>En caso de seleccionar la opción 2) Salir se finaliza la ejecución del programa mostrando al usuario un mensaje: "Se ha cerrado el programa".</p>
<p>En caso de ingresar cualquier otra opción diferente de 1 o 2 se mostrará al usuario un mensaje indicando el error: "Opción inválida. Ingrese un número de las opciones listadas."</p>
