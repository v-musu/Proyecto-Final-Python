<h1>Sobre el proyecto</h1>
<h2>Resumen</h2>
<p>Se trata de un proyecto integrador desarrollado exclusivamente en lenguaje Python con el objetivo de aplicar los conocimientos obtenidos para acceder a la certificación de finalización del curso brindado por ADA IT.</p>
<p>El programa consiste en un generador y editor de catálogos de películas almacenados en archivos .txt, dónde cada archivo se corresponde con un catálogo compuesto por un conjunto de datos de una o más películas. El identificador del catálogo es su nombre y cada película almacenada detalla: título, director, año y review (crítica).</p>
<p>Hasta el momento el programa no cuenta con una interfaz gráfica para su interacción, aunque se planea desarrollar una en el futuro en una nueva versión del mismo.</p>
